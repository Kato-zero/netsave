# 🎬 Tube Download

YouTube Video & MP3 Downloader using Node.js and yt-dlp.

## Features

- Download YouTube videos
- Convert to MP3
- Download MP4 720p
- Download MP4 1080p
- Simple frontend
- Fast Node.js backend

---

## Requirements

Install:

- Node.js
- yt-dlp
- ffmpeg
